### 建構子特點：
1. 沒有 Output 
2. 名稱與Class 相同
3. 屬於 Method 中的一種

### 使用目的：
1. 減少重複程式碼撰寫
2. 可以善用C# Overloaded多載屬性 
3. 在建立物件時就會產生，因此我們可以用來賦予初始值

範例：
1. 先建立一個class
```C#
namespace demo
{
  class Student
  {
   public int grade;
   public string name;
   public string studentID;

    //建構子
    public Student()
    {
       grade = 2;
    }
    
    public Student(int grade1, string name2)
    {
       grade = grade1;
       name = name2;
    }

  }

}

```
2. 在建立一個新的物件
```C#
using demo.Models
public class Demo
{
   new Student(); 
   //產生grade = 2;

   new Student(1,"2");
   //產生grade = 1;name ="2"
}

```
