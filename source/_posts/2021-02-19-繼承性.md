### 繼承性
可以繼承其他物件屬性和方法。其目的在於減少重複程式碼撰寫，方便程式維護。

### 範例
```c#
public class Demo
{
  private int hp;
  public string getHp()
  {
     return hp;
  }
}
             //繼承Demo屬性和方法
public class Demo2: Demo
{
   private int hurt;
}

public class Test
{
  Demo2 A = new Demo2();
  A.int = 10;
  Console.WriteLine(A.int); //10
}

```
